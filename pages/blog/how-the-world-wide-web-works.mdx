# How The World Wide Web Works

Wanting to become a Web Developer is a great challenge to take on. I'm hoping I can help make that journey a little easier. Today we will be going over part one of my Modern Web Development series where we will discuss the basics of web development theory.

We will discuss:

1. What the Web (or World Wide Web)
2. What is the Internet?
3. What is the browser?
4. How does a browser display a website?
5. What is an IP address?
6. What is HTML, CSS and Javascript?
